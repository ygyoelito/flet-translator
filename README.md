## Description:
Text translator into several languages using the Microsoft Azure AI Translator service


## Installation

Install the python modules using requirements.txt

```bash
pip install -r requirements.txt
```

Create the .env file with the following keys:
```bash
KEY=
ENDPOINT=
LOCATION=
```
**Tip**: The values must obtain them from its application of **[Microsoft Azure](https://azure.microsoft.com/)**   


Execute the application with:
```bash
flet run app.py
```
